Markdown中使用html标签

发表于 2018-08-28 更新于 2019-10-08 分类于 工具使用 ， Markdown 阅读次数：
本文字数： 376 阅读时长 ≈ 1 分钟

有时候,我们会在笔记中使用一些HTML标签,但是Markdown中直接使用html标签会被解析为格式.该怎么书写，可以直接显示为html的标签呢.

问题出现原因：

Markdown支持html语法，所以会对相应的标签进行解析。

问题解决办法：

  1. 使用引号把标签包起来

<h1>这是一个标题</h1>

  1. 使用转义字符来表示相应的标签

<h1>这是一个标题</h1>
