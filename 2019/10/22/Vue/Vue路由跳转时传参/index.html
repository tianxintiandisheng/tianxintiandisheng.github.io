本文主要介绍了vue中使用路由进行页面的跳转时,vue的路由如何传递参数,第二个页面如何获取参数.

一. 通过router-link进行跳转

  1. path -> 是要跳转的路由路径,也可以是路由文件里面配置的 name 值,两者都可以进行路由导航
  2. params -> 是要传送的参数,参数可以直接key:value形式传递
  3. query -> 是通过 url 来传递参数的同样是key:value形式传递
